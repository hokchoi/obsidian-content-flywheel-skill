# Vault Schema

This skill works with any Obsidian vault. Prefer adapting to the user's existing structure.

## Generic Structure

```text
00_Workflow/
01_Inbox/
02_Knowledge_Atoms/
03_Topic_Seeds/
04_COPE_Drafts/
99_Templates/
```

## Chinese Content System Structure

If the vault already has these folders, use them:

```text
00_中控台/
02_输入库/
03_选题池/
04_内容生产/
05_素材生产/
08_AI工作流/
99_模板/
```

## Frontmatter Fields

Common fields:

- `type`
- `status`
- `source_url`
- `source_notes`
- `related_notes`
- `platform_fit`
- `created_date`
- `tags`

Use the user's existing frontmatter style when visible.
